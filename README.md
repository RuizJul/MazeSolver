This project consists of the development of a maze-solving game in Java, where an intelligent agent must find the optimal path to escape a maze by using tree-based data structures and the Minimax algorithm for decision-making.

The maze is modeled as a state tree, where each node represents a possible position of the player inside the maze, and each edge represents a valid action or movement. Based on this tree, the Minimax algorithm evaluates different possible paths by considering favorable and unfavorable states, aiming to optimize the agentâ€™s decisions and select the most suitable route.
